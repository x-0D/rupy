# rupy
Пишите программы на русском синтаксисе python!

## Установка

$　pip install -U rupy

## Использование

После установки интерпретатор python доступен сразу. Попробуйте запустить его сразу!

```
印出 '哈囉, 世界'
```

```
$ python interpreter.py
zhpy X.X in darwin on top of Python 2.X.X
>>> 印出 '哈囉, 世界'
哈囉, 世界
```

這樣你就完成了你的第一個中文程式囉!
